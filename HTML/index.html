HTML

My adventure started with HTML, the skeleton of web design. Diving into HTML felt like learning a new language, one that enabled me to communicate my ideas through the structure of web pages. Creating my first "Hello, World!" page was a simple act, but it ignited a sense of achievement and wonder within me. However, as I delved deeper, I encountered the complexities of elements, attributes, and semantic web principles. Despite the daunting challenges, my curiosity morphed into a determination to understand and adeptly apply these concepts, leading me to craft my first fully structured web page. There is not much that can be done alone with HTML, as it is just the backbone language of creating a website. However, throughout this page I will display some of the many things that can be done with HTML.
