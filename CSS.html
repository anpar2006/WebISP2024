CSS

Once I became comfortable with HTML, I realized that structure alone wasn't enough to capture the full potential of web design. That's when CSS came into play - the style and beauty behind the web. Learning CSS was akin to discovering an array of colors and learning how to paint; it transformed my basic web pages into visually appealing masterpieces. I experimented with colors, fonts, and layouts, tackling the challenges of responsiveness and cross-browser compatibility head-on. The journey through CSS was filled with trials and errors, especially when grappling with the intricacies of positioning and the box model. Yet, each challenge served as a valuable lesson, sharpening my skills to create websites that were not only functional but aesthetically pleasing as well. Throughout this page I will display some of the many things that can be done with CSS.
